Privacy Policy for KanActive AI Lite

Last updated: July 15, 2025

KanActive AI Lite ("the Extension") is a Chrome extension developed to help users detect and prevent accidental submission of sensitive data — specifically PCI (credit card) data — into ChatGPT. This policy outlines how the Extension handles data and user privacy.

1. Data Collection

We do not collect, transmit, or store any user data externally. All detection, analysis, and keyword monitoring is performed locally within your browser. The Extension does not:

  • Transmit any data to external servers
  • Use analytics tools
  • Sync data between devices
  • Access or collect personally identifiable information (PII) beyond the scope of the current Chrome session

2. Permissions Justification

To function as intended, the Extension uses the following Chrome permissions:

  • activeTab: Allows the extension to interact with the current ChatGPT tab to scan inputs for sensitive content.
  • host permissions (https://chatgpt.com/*): Enables scanning of user input fields within ChatGPT.
  • storage: Used exclusively for local storage of detection preferences and detection logs.

3. Data Storage

All logs and settings (including monitored keywords and detection history) are stored using Chrome's storage.local API. This means the data:

  • Remains on your device
  • Is never sent to the developer or any third party
  • Can be cleared by the user at any time by closing the browser

4. Remote Code & APIs

The Lite version of KanActive AI does not execute or load remote code. All code is bundled within the extension at install time. The extension does not communicate with external APIs except for optional OAuth use to identify the current Chrome profile (if identity is enabled), which remains local.

5. User Control

You may uninstall the Extension at any time. You may also clear any stored settings or detection logs by closing the browser. No user account or signup is required.

6. Changes to This Policy

If this privacy policy is updated in the future (e.g., for a paid or enterprise version), you will be notified via the Chrome Web Store and this policy will be updated accordingly.

7. Contact

If you have any questions or concerns about this privacy policy, you can contact the developer using this web form.
